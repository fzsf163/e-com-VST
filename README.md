# Shadcn Admin Dashboard

Admin Dashboard UI crafted with Shadcn and Vite TanStack Router

Clone the project

```bash
  git clone https://github.com/fzsf163/e-com-VST.git
```

Go to the project directory

```bash
  cd shadcn-admin
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run dev
```
